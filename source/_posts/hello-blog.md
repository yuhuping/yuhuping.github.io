欢迎来到我的个人博客！本站基于 [Hexo](https://hexo.io/) + [Butterfly](https://butterfly.js.org/) 主题搭建，托管在 GitHub Pages，每次 `git push` 自动构建部署。

## 常用命令

```bash
hexo new "文章标题"   # 新建文章，生成在 source/_posts/
hexo server           # 本地预览 http://localhost:4000
hexo clean            # 清理缓存
git add . && git commit -m "post" && git push   # 推送即自动部署
```

## 接下来

- 修改 `_config.yml` 里的站点标题、作者
- 在 `_config.butterfly.yml` 调整主题：导航、社交链接、首页背景等
- 开始写第一篇真正属于你的文章 ✍️
